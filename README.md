# Databricks Training — Data Engineer Associate Career Path

Interactive starter project to onboard new colleagues onto Databricks and prepare
them for the **Databricks Certified Data Engineer Associate** certification
(May 2026 syllabus).

Participants start from this repo and, guided by the project guide and TODO
requirements, build a full **medallion-architecture** solution (bronze → silver →
gold) over a synthetic **insurance** dataset, then publish dashboards answering the
business questions in the requirements doc.

> 🚧 Planning in progress — structure and guide documents are being authored.
